1.大驼峰命名法（帕斯卡命名法）
命名规则：每个单词的第一个字母都要大写
例如：
int MyAge;
char MyName[10];
float ManHeight;
function Name;
int Wang;


2.小驼峰命名法
命名规则：要求第一个单词首字母小写，后面其他单词首字母大写
例如：
int myAge;
char myName[10];
float manHeight;
function myName;
int wangJiachen;

3.匈牙利命名法
命名规则：变量名=属性+类型+对象描述
即通过在变量名前面加上相应的小写字母的符号标识作为前缀，标识出变量的作用域，类型等。要求前缀字母用变量类型的缩写，其余部分用变量的英文或英文的缩写，单词第一个字母大写。
例如：
iMyAge（“i”指int）
cMyName（“c”指char）
fManHeight（“f”指float）
fnHandler（“fn”指函数）
oDivl（“o”指对象）

4.下划线命名法
命名规则：要求单词与单词之间通过下划线连接即可
例如：
int my_age;
char my_name[10];
float man_height;
string user_name;
show_message
